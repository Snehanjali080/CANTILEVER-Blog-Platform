# CANTILEVER Blog Platform

A full-stack blog application built using the MERN Stack that allows users to create, read, update, and interact with blog posts.

## Features

* User Authentication (JWT)
* Create Blog Posts
* Edit and Delete Own Posts
* Like / Unlike Posts
* Comment System
* Notifications
* Trending Blogs
* View Counter
* Search Functionality
* User Profiles
* Admin Controls

## Tech Stack

### Frontend

* React.js
* React Router
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
